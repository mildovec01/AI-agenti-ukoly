# LLM + Tools Demo

Tento projekt ukazuje jednoduché použití OpenAI LLM modelu s nástroji
(function calling). Model:

1. Dostane uživatelský dotaz (např. „Vypočítej: a daný příklad“).
2. Rozhodne se zavolat nástroj, podle daného promptu.
3. Python skript nástroj dále poté lokálně provede.
4. Výsledek nástroje pošle zpět modelu, který ho vysvětlí v přirozeném jazyce.
